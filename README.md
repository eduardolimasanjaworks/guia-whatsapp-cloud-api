# Guia — WhatsApp Cloud API (Meta)

Documentação em português claro para cadastro e uso da API oficial do WhatsApp.

## Arquivos

| Arquivo | Descrição |
|---------|-----------|
| `guia-visual-api-oficial.html` | Guia interativo (abrir no navegador) |
| `guia-whatsapp-cloud-api-credenciais.md` | Checklist técnico: IDs, token, links Meta |

## Como usar

1. Abra `guia-visual-api-oficial.html` no navegador (duplo clique ou arrastar para o Chrome).
2. Leia o modal inicial e siga as seções na ordem.
3. Use o `.md` como referência rápida ao copiar credenciais.

## Observação

Não commite arquivos `.env` com tokens reais. O guia só explica o processo; credenciais ficam no seu ambiente seguro.
